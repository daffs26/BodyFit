# BodyFit — Gym Progress Tracker 💪

A mobile-first Progressive Web App (PWA) to track your gym progress, nutrition, and achieve your fitness goals — **100% local, no account needed.**

## Features

- 🏋️ **Workout Logger** — Log sets, reps, and weight with progressive overload tracking
- 🥗 **Nutrition Tracker** — Track macros (protein, carbs, fat) with TDEE calculator for bulking/cutting
- 📊 **Analytics** — Weight trend charts, weekly volume charts, and 1RM calculator  
- 📅 **Training Planner** — Monthly goals with Push-Pull-Legs, Upper-Lower, Full Body, and Bro Split programs
- 📚 **Exercise Library** — 40+ exercises organized by muscle group with custom exercise support
- 🔒 **100% Local** — All data stored on-device via IndexedDB (Dexie.js), export/import as JSON

## Tech Stack

- **React 18** + **Vite** — Fast, production-ready build
- **Zustand** — Lightweight state management
- **Dexie.js** — IndexedDB wrapper for local persistence
- **Framer Motion** — Smooth mobile-like transitions
- **Chart.js** — Interactive charts for progress tracking
- **Lucide React** — Clean icon library

## Color Theme

Black `#0B0B0C` × White `#FFFFFF` × Orange `#FF6B00`

## Getting Started

```bash
npm install
npm run dev
```
